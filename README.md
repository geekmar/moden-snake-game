# 🐍 现代化贪吃蛇小游戏

一个基于 HTML5 Canvas 的现代化贪吃蛇游戏，界面简洁、动作灵敏，支持键盘控制与暂停功能。

## ✨ 功能特点
- **现代 UI**：玻璃拟态风格 + 渐变色 UI
- **流畅动作**：requestAnimationFrame 驱动，响应快
- **动态加速**：每吃 5 个食物速度提升
- **分数记录**：本地存储最高分
- **键盘支持**：方向键 / WASD 控制
- **暂停/继续**：空格键或按钮暂停

## 🚀 快速开始
1. 将项目保存为 `snake.html`
2. 双击打开即可运行（无需服务器）

## 🎮 操作说明
| 操作 | 功能 |
|------|------|
| ↑ ↓ ← → / WASD | 移动 |
| 空格 | 暂停 / 继续 |
| Enter | 重新开始 |
| 按钮 | 开始/暂停 |
